高光谱图像变化检测研究方法

发表于2023-02-23|更新于2023-02-23|深度学习
|阅读量:

概述

分类

按应用目的：异常，多类，时间序列，二分类（现在的文章基本都是这个）

按训练数据：有监督，半监督，无监督

自动化程度：全自动、半自动和手动

ANOMALOUS CHANGE DETECTION

一般而言，HS图像的ACD方法可以分为单实例目标检测和多实例目标检测。通常特别注意检测由于制造的小物体的插入、删除或移动以及小物体的插入、删除或移动而产生的微小变化

BINARY CHANGE DETECTION

简化为一个二分类问题，即判断每个像素是否发生了变化。特点是：1简单易实现，BCD只需要两幅多时相高光谱图像作为输入，不需要先验知识或训练数据。2高效快速：BCD只需要对每个像素进行一次判断，不需要考虑变化的类型或程度。3适用范围广，BCD可以应用于各种场景和目标，如土地覆盖、建筑物、水体等。

MULTICLASS CHANGE DETECTION

指在双时相或多时相高光谱图像中，不仅要检测出发生变化的区域，还要识别出变化的类型。例如，在城市扩张的过程中，可能会有农田、森林、水体等不同类型的地物被建筑物所取代。多类变化检测可以提供更丰富和更细致的变化信息，有助于监测和评估环境和社会经济的影响。

CHANGE DETECTION IN IMAGE TIME SERIES

分析和监测地表变化的过程和趋势。例如，在森林火灾、城市扩张、农业生产等领域，图像时间序列变化检测可以提供重要的信息和指标。图像时间序列变化检测可以分为两类：一类是基于成对图像比较（pairwise image comparisons）的方法，即在两个时期之间进行变化检测；另一类是基于多个图像比较（multiple image comparisons）的方法，即在一个时间段内进行变化检测。

一般步骤

高光谱数据预处理：主要是对双时序图像进行配准，对辐射进行校正，消除光和大气对高光谱图像的影响。

选择变化检测方法：最重要的一步。

结果和评估：变化检测的结果通常以两种形式显示：不变的像素和变化的像素。（就是上面说的BINARY CHANGE DETECTION）

存在的挑战

  • 数据维度过高：是所有HS图像处理任务的常见问题。增加了CD的计算开销和内存需求。需要高效的数据处理和分析方法。

  • 数据冗余：图像具有丰富的波段，不同的波段对变化检测的贡献不同。对变化检测贡献不大的波段可能会影响最终的检测精度，数据量过大，会导致计算机计算压力大幅增加。然而，目前提出的用于变化检测的波段选择方法仍然很少。

  • 如何利用通道优势

  • 噪声干扰：

    • 高光谱图像受到大气、仪器和辐射等因素的影响，导致噪声和失真。
    • 当光谱分辨率提高时，传感器处测量的反射率降低，而与信号无关的噪声不会降低，从而降低信噪比。
    • 当频带数目增加到一定限度以上时，HS频段所代表的信息变得更加稀疏和明显，这可能会降低ACD的区分能力（特别是在识别多类变化）。

  • 缺少标签：(Multilayer Cascade Screening Strategy for Semi-Supervised Change Detection in Hyperspectral Images设计了一种半监督训练方法。下面还有一篇使用无监督特征提取网络，少量标签训练分类器）

  • 光谱差异：多时相高光谱图像之间存在光谱差异，如光照变化、观测角度变化和植被季节性变化等，影响CD的准确性。

CSANet: Cross-Temporal Interaction Symmetric Attention Network for Hyperspectral Image Change Detection

该框架主要包括三个对称的阶段，即2-D孪生Conv骨干、CSA特征增强和多时相特征融合。

首先，将成对的HS图像送入孪生2-D CNN主干，以提取HS图像的浅层特征。

然后，通过将CSA引入到网络中，实现了空间-光谱-时间联合特征的融合和增强。

在最后一阶段，生成双时相HS图像的差异特征，以获得像素级的CD结果。

主要贡献

（其实他的创新点只有CSA这一点）

  • 提出了一种新的跨时间交互对称注意算法CSA。在CSA中，结合自注意来增强每个时间图像的特征表示能力，并利用跨时间注意来整合各个时间特征嵌入的不同特征
  • 将CSA引入孪生2-D CNN结构中，构建了HS图像CD的CSANet。CSANet能够有效地提取和整合HS图像的时空-光谱-时间联合特征，同时增强对变化的特征分辨能力。
  • 与其他模型相比有提升

A CNN Framework With Slow-Fast Band Selection and Feature Fusion Grouping for Hyperspectral Image Change Detection

T1和T2表示双时相高光谱遥感图像。SFBS方法用于提取T1和T2中对变化检测贡献最大的波段集。

为了充分利用高光谱图像的通道优势，采用了构造差分矩阵的方法。然后使用自定义卷积网络层来提取和级联差分矩阵的变化信息特征图。

将特征级联的输出连接到全连通层，将全连通层的所有节点分成k组，每组包含m个节点，从而生成更具方向性、更有利于变化检测的特征组。在生成分组后，连接Softmax分类器，最终得到变化检测二值图像。

主要贡献

1)为解决波段冗余问题，在慢特征分析(SFA)的基础上，提出了一种SFBS方法，通过选择慢特征带和快特征带来更好地提取变化特征和不变特征，更有效地分离变化像素和不变像素;

2)利用差分矩阵丰富变化信息层次，为变化检测提供更多的变化特征，便于变化信息特征的提取

3)采用FFG方法生成更具鉴别性的特征组，并设计了相关损失函数。在多个真实高光谱数据集上的实验结果表明，SFBS可以降低计算机的运行负荷，提高变化检测的精度，FFG也可以提高变化检测的精度。

A Spectral–Spatial Change Detection Method Based on Simplified 3-D Convolutional Autoencoder for Multitemporal Hyperspectral Images

首先通过训练的无监督简化的3D卷积autoencoder自动提取空间光谱特征，然后将提取的两个特征地图沿光谱维度进行叠加。最后，使用加入二维卷积层的Softmax分类器来获得检测结果。

整个框架是半监督的，只需要几个标记样本来训练分类器（上图STEP2右边部分）。

主要贡献

  • 采用深卷积自适应学习的方法，通过无监督学习来提取联合空间-谱特征，更适合处理变化的场景
  • 通过增加1×1×L的简化卷积核和去掉合并层，设计了一种简化的3DCAE。简化版本可以减少数据的冗余度，并且参数比标准核少。
  • 整个框架是半监督的，只需要几个标记样本来训练分类器。由于HIS的标签获取费时费力，因此该方法可以减少对标签数据的依赖。

CDFormer: A Hyperspectral Image Change Detection Method Based on Transformer Encoders

首先，将HIS的两个面片分别展平为像素序列。然后，PE模块将序列转换为嵌入序列，并向该序列添加空间和时间信息。然后，将该序列送入变换编码模块，以提取联合加权的空间-频谱-时间特征。最后，利用全连通网络生成检测结果。（感觉就是直接套ViT…）

主要贡献

  • 该方法具有全局时空感受野，能够同时捕获空间和时间信息。
  • 在HSI CD中首次引入了变压器编码器。
  • 该方法只需要少量的标注样本就可以训练网络。有利于降低样品贴标成本，提高自动光盘制作的可行性
